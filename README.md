# La Página Hueb

Un pequeño proyecto web sobre cuestiones filosóficas, existenciales y del día a día.

## Qué hace

- Muestra contenido relacionado con la vida, el tiempo y la reflexión.
- Puede consultar datos externos como el tiempo y las horas de luz (¡e incluso más si el tiempo lo permite!).
- Guarda alguna información local en el navegador o no.

## Tecnologías

- HTML
- CSS
- JavaScript
